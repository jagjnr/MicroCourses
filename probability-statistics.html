Probability and Statistics

Gain foundational knowledge in probability theory and statistical analysis. This course covers essential concepts such as random variables, probability distributions, statistical inference, hypothesis testing, regression analysis, and data visualization. Develop practical skills to analyze data and make data-driven decisions using statistical methods.

Modules

  • Module 1: Introduction to Probability Theory
  • Module 2: Random Variables and Distributions
  • Module 3: Descriptive Statistics and Data Visualization
  • Module 4: Inferential Statistics and Hypothesis Testing
  • Module 5: Correlation and Regression Analysis
  • Module 6: Probability Distributions and Sampling
  • Module 7: Confidence Intervals and Estimation
  • Module 8: Bayesian Inference and Applications
  • Module 9: Statistical Modeling Techniques
  • Module 10: Real-World Applications of Statistics
